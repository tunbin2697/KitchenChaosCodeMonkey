# Kitchen Chaos - Unity Game

This is my first Unity game development project following the [CodeMonkey Kitchen Chaos tutorial](https://www.youtube.com/watch?v=AmGSEH7QcDg).

## About

Kitchen Chaos is a cooking simulation game where players prepare and serve food orders in a chaotic kitchen environment. This project is being built following CodeMonkey's comprehensive Unity tutorial series.

## Unity Version

- Unity 2022.3.10f1 (LTS)
- Universal Render Pipeline (URP)
- New Input System

## Project Structure

```
KitchenChaosCodeMonkey/
├── Assets/
│   ├── Scenes/          # Game scenes
│   ├── Scripts/         # C# scripts
│   ├── Prefabs/         # Reusable game objects
│   ├── Materials/       # Materials and shaders
│   ├── Textures/        # Textures and sprites
│   └── Audio/           # Sound effects and music
├── ProjectSettings/     # Unity project settings
└── Packages/           # Package dependencies
```

## Getting Started

1. Clone this repository
2. Open the project in Unity 2022.3.10f1 or later
3. Open the main scene in Assets/Scenes/
4. Press Play to run the game
